# Retail-Sales-Analytics-Project
This project analyzes retail sales data to understand customer behaviour, sales trends, and marketing effectiveness.

## Overview

The main objective of this project is to analyze sales performance, identify high-value customers, study customer response patterns and provide business recommendations using data analytics tools like 
Python, SQL, Excel, EDA.

## Tools & Technologies

-   Python
-   Pandas
-   Seaborn
-   Matplotlib
-   Excel

## Project Objectives

-   Analyze customer purchase behaviour
-   Identify high value customers
-   Evaluate sales trends
-   Generate business recommendations

## Dataset

Two datasets were used: 1. Transaction Dataset 2. Customer Response Dataset

## Key Analysis Performed

-   Exploratory Data Analysis (EDA)
-   Customer Segmentation (RFM Analysis)
-   Time Series Sales Trend Analysis
-   Churn Analysis

## Key Insights

-   Seasonal sales variations exist
-   High-value customers contribute significant revenue
-   Marketing responses impact customer spending behaviour

## Business Recommendations

-   Implement loyalty programs
-   Improve targeted marketing campaigns
-   Plan inventory based on seasonal trends
